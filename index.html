Generative AI based Geometry Generation for Computational Fluid Dynamics
Thesis Project

Master's Thesis work done at TUM in collaboration with BMW. This work aims on using Generative Models to compute initial guesses for 2D and 3D flow manifold geometries that minimize pressure drop between inflow and outflow. My Academic supervisor was Prof. Nils Thuerey.

Solving Kuramoto Sivashinsky Equation using Graph Convolutional Neural Networks
Project

This project involved using Message Passing Graph Neural Networks and Convolutional Neural Networks for prediction solutions to Kuramoto Sivashinsky Equation, which is a fourth order stiff ODE. Both networks were implemented in a ResNet like fashion and various temporal unrolling techniques were explored to test for solution prediction and correction. This work was done in association with Thurey's group at TUM chair of Graphics and Visualization

Dense Point Cloud Prediction from a Single RGB Image
Project

Here a Point Set Generation Network, a two-stage reconstruction pipeline that takes a single RGB image of an 3D object as input and predicted a high-quality point cloud representing the shape via an Encoder-Predictor and a folding Decoder network. This group project was done as a part of the Machine Learning for 3D geometry course held by Prof. Dr. Angela Dai.

3D Shape Reconstruction using Deep-SDF
Project

Implemented a Deep-SDF architecture that learns continuous representation of shapes. Trained it on the shapenet dataset to achieve high resolution shape completion and latent interpolation on the test set. Also done as a part of the Machine Learning for 3D geometry course held by Prof. Dr. Angela Dai.

Pedestrian Distribution Learning using VAEs
Project

Implemented and trained a Variational Auto-Encoder to learn pedestrian distribution inside the TUM Mathematik/Informatik building at Garching. After learning distribution, estimation of pedestrians in a given area using the trained decoder was carried out for a hypothetical fire evacuation scenario.
